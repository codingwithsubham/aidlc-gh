# User Authentication & Management

**Long Description**: This epic will implement a comprehensive user authentication system that allows both customers and sellers to register, log in, log out, and manage their profiles. It includes a registration process with validation such as email uniqueness and password strength, ensuring security and a good user experience.

**Acceptance Criteria**:
- Users can register as either customers or sellers with mandatory fields.
- Unique email validation during registration.
- JWT-based authentication for secure login/logout.
- Password recovery features are implemented.

**Criticality**: High

**Dependency Priority**: None
